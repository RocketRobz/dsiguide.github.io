    Instructions

    Section I - Formatting Your SD Card (Windows)
    1. Open up the 'Computer' window by pressing Windows key + E on your keyboard.
    2. Your SD card should be displayed in the 'Devices with Removable Storage' section. Right-click on your SD card to bring up the right-click menu options. Select Format.
    3. In the Format window make sure that the correct File System is selected. It should be set to FAT32
    4. Select start, and wait for it to finish.
    5. Copy the necessary files over again and try the exploit again.
    Section II - Using the exploit
    1. Find the page on the site with installation (https://www.dsiguide.me/downgrading)
    2. Folow the instructions EXACTLY.
    3. If you still have issues, clear all your flipnote studio data (not the app itself!) and try again.
    4. If needed, contact the DSi Homebrew discord.
